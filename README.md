# 📊 HR & Workforce Analytics Dashboard

## Overview
An interactive HR dashboard built entirely in Microsoft Excel 
analyzing 1,470 employee records across 3 departments.

Built to answer one core business question:
**Why are we losing employees and what can we do about it?**

## 📈 Key Metrics
| Metric | Value |
|---|---|
| Total Employees | 1,470 |
| Attrition Rate | 16% |
| Average Monthly Salary | $6,503 |
| Average Age | 36.9 years |

## 🔍 Key Findings
- Employees aged 20-30 have the highest attrition rate
- Sales pays the most but still has highest attrition
- Gender split: 60% Male, 40% Female
- Attrition is a culture problem, not a salary problem
